# REST API - Produtos

Esta é uma aplicação CRUD (_Create, Read, Update and Delete_) que simula um cadastro de produtos e foi desenvolvida de forma guiada durante a playlist de vídeos sobre **API REST com Spring Boot** da [Michelli Brito](https://www.youtube.com/c/MichelliBrito) para fixação dos conceitos de REST API, Spring Boot e Spring MVC.

### Bibliotecas utilizadas

* [Official Apache Maven](https://maven.apache.org/guides/index.html)
* [Spring Boot](https://docs.spring.io/spring-boot/docs/2.5.3/maven-plugin/reference/html/)
* [Spring Web](https://docs.spring.io/spring-boot/docs/2.5.3/reference/htmlsingle/#boot-features-developing-web-applications)
* [Spring Data JPA](https://docs.spring.io/spring-boot/docs/2.5.3/reference/htmlsingle/#boot-features-jpa-and-spring-data)
* [Spring Boot DevTools](https://docs.spring.io/spring-boot/docs/2.5.3/reference/htmlsingle/#using-boot-devtools)
* [Swagger UI 2](https://swagger.io/tools/swagger-ui/)
